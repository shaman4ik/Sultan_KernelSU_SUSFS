---

## Причина — дрейф внешнего репо, НЕ мой откат

### Фатальная ошибка (susfs-шаг `sultan.yml`, KSU-Next glue)
```
cp: cannot stat '.../kernel_patches/wild/ksun-293ca01-susfs-v2.1.0-a14-6.1-ef16cbce.patch':
No such file or directory   →   exit 1
```
`TheWildJames/kernel_patches/wild/` больше **не содержит** запиненный glue-патч.
Сейчас там лежит **`ksun-5a4a718-susfs-f7ae19ef-gki-android14-6.1.patch`** — другой пин
(KSU-Next `5a4a718` + susfs `f7ae19ef`). А `sultan.yml` жёстко тянет старое имя
`ksun-293ca01-…-ef16cbce` и чекаутит susfs `ef16cbce`.

Итог: **связка KSU-Next + v2.1.0 в текущих пинах больше не воспроизводится из апстрима**
(`cp` без `|| true` → падение job).

### Вторичное (проглочено `|| true`, на падение не влияет)
`50_add_susfs` (v2.1.0) не лёг на `kernel/sys.c` Hunk #3 → `sys.c.rej` — это хунк
uname-спуфа в кастомном `newuname` kerneltoast. Единственный артефакт прогона —
`rej-gs201-wksu-susfs.zip` (только `sys.c` + `sys.c.rej`). **Ядро не собрано.**

### Про мой Стадия-0-откат
К падению отношения не имеет — он лишь продвинул сборку до этого места (раньше job
падал раньше, на `sys.c_fix.patch`). `trigger-release` ожидаемо **не запускался**
(job-зависимость + guard) — релиз/тег не публиковались, artifact-only удержан.

---

## Что это значит для «фоллбэка»

Запиненный baseline-стек (KSU-Next + v2.1.0 через TheWildJames) **как есть не собирается**
из-за апстрим-дрейфа. А Стадия 1 (Replace) этот glue (`ksun-susfs` + `setup.sh` KSU-Next +
патчи TheWildJames) **целиком выпиливает** и ставит ReSukiSU + вендорный `50_/51_` (v2.0.0).

---

## Решение (значимая развилка — за человеком)

- **A (рекомендую): не чинить стек, который мы удаляем.**
  Фоллбэк-точка = чистый исходный коммит `af9cb77` (source-level, без собранного zip),
  baseline-артефакт-сборку пропускаем, идём в Стадию 1 Replace. Реальная валидация — на CP5.

- **B: починить baseline** — поднять пины glue+susfs до текущих апстрим
  (`ksun-5a4a718` / susfs `f7ae19ef`) в `sultan.yml`, перезапустить, получить реально
  собранный KSU-Next-фоллбэк. Больше работы ради фоллбэка, который, скорее всего, не понадобится.
